# 产品需求文档 (PRD)

## 项目名称
Intestine ASSistant - 肠道健康助手

## 1. 产品概述

### 1.1 产品背景
肠道健康是人体健康的重要组成部分，排便情况是反映肠道健康的重要指标。通过记录和分析排便数据，用户可以更好地了解自己的肠道健康状况，及时发现潜在问题。

### 1.2 产品定位
一款面向关注肠道健康用户的移动端应用，帮助用户便捷记录排便数据，并通过AI分析提供健康建议。

### 1.3 目标用户
- 关注肠道健康的普通用户
- 有消化系统疾病史需要监测的用户
- 健康管理爱好者
- 医疗健康领域从业者

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 排便时间记录
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 手动输入 | 用户可手动输入排便日期和时间 | P0 |
| 计时器功能 | 提供计时器，用户可实时记录排便时长 | P0 |
| 时间编辑 | 支持对已记录的时间进行修改 | P1 |
| 历史记录 | 查看历史排便时间记录 | P0 |

#### 2.1.2 排便情况记录（选项卡式）
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 粪便形态 | 基于布里斯托大便分类法（7类）选择 | P0 |
| 颜色记录 | 选择粪便颜色（正常/异常色系） | P0 |
| 气味评估 | 主观评估气味程度 | P1 |
| 排便感受 | 记录排便时的感受（顺畅/困难/疼痛等） | P0 |
| 伴随症状 | 记录腹痛、腹胀等伴随症状 | P1 |
| 备注 | 自由文本备注 | P2 |

#### 2.1.3 AI数据分析
| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 数据统计 | 统计排便频率、时间分布等 | P0 |
| 趋势分析 | 分析排便习惯变化趋势 | P0 |
| 健康评估 | 基于数据给出肠道健康评分 | P1 |
| 异常预警 | 识别异常模式并提醒用户 | P1 |
| 健康建议 | 提供个性化的饮食和生活建议 | P1 |
| 报告生成 | 生成可分享的健康报告 | P2 |

### 2.2 辅助功能

#### 2.2.1 用户管理

#### 2.2.2 数据管理
- 数据导出（CSV/PDF）

#### 2.2.3 提醒功能
- 定时提醒记录
- 久未排便提醒

## 3. 非功能需求

### 3.1 性能需求
| 指标 | 要求 |
|------|------|
| 应用启动时间 | < 3秒 |
| 页面加载时间 | < 1秒 |
| 数据保存响应 | < 500ms |
| AI分析响应 | < 5秒 |

### 3.2 安全需求
- 用户数据加密存储
- 传输过程HTTPS加密
- 符合数据隐私保护法规
- 支持数据本地存储选项

### 3.3 可用性需求
- 界面简洁直观
- 操作步骤最少化
- 支持离线使用
- 多语言支持（中文/英文）

### 3.4 兼容性需求
- iOS 13.0+
- Android 8.0+
- 支持主流屏幕尺寸

## 4. 用户界面设计

### 4.1 主要页面
1. **首页** - 快速记录入口、今日概览
2. **记录页** - 时间输入、选项卡式情况记录
3. **统计页** - 数据可视化图表
4. **分析页** - AI分析结果、健康建议
5. **我的页** - 个人设置、数据管理

### 4.2 设计原则
- 简洁明了，减少用户认知负担
- 色彩温和，避免引起不适
- 图标直观，便于快速识别
- 交互流畅，提升使用体验

## 5. 数据指标

### 5.1 布里斯托大便分类法
| 类型 | 描述 | 健康状态 |
|------|------|----------|
| 类型1 | 分离的硬块状 | 便秘 |
| 类型2 | 香肠状但结块 | 轻度便秘 |
| 类型3 | 香肠状表面有裂纹 | 正常 |
| 类型4 | 香肠状光滑柔软 | 正常（理想） |
| 类型5 | 柔软断块状 | 缺乏膳食纤维 |
| 类型6 | 糊状松散 | 轻度腹泻 |
| 类型7 | 完全液体 | 腹泻 |

## 7. 风险评估

| 风险 | 影响 | 应对措施 |
|------|------|----------|
| 数据隐私泄露 | 高 | 加密存储、本地优先策略 |
| AI建议准确性 | 中 | 明确免责声明、持续优化模型 |
| 用户使用频率低 | 中 | 提醒功能、游戏化激励 |
| 法规合规风险 | 高 | 咨询法务、符合医疗应用规范 |
